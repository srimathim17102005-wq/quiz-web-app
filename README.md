# Quiz Web Application

A full-stack quiz web application built using Flask.

## 🚀 Features
- Multiple choice questions
- Score calculation
- Database integration (SQLite)
- Dynamic result display

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite

## ▶️ How to Run
1. Clone the repo
2. Install requirements
3. Run app.py
